\azfact:

Format: \stmt{azfact $<$literal$>$}

Options: $<$literal$>$ the fact

Effect: The fact \stmt{(hn literal)} will be inserted at the end of the (possibly empty) 
	database \stmt{*factbase*}. 
	This command is an abbreviation of \stmt{az (fact literal)}.

see also: \az, \consult, \destroy, \replace
