\stmb{replace-strategies}:

Format: \stmt{replace-strategies $<$filename$>$}

Options: $<$filename$>$ a stringified or normal pathname

Effect: Replacing the database \stmt{*fc-strategies*} with the strategies for forward 
	chaining from file $<$filename$>$. If no argument is given the default 
	strategies are consulted. 
	If no extension is provided, \COLAB{} extends the filename with \stmt{".rf"}.

see also: \consult, \destroy, \replace
