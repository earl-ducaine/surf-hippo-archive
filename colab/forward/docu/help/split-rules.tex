\stmb{split-rules}:

Format: \stmb{split-rules}

Options: none

Effect: All up-clauses of the database \stmt{*rule-database*} are copied to the 
	\stmt{*up-rulebase*}, and all rl-clauses are copied to the database 
	\stmt{*hn-rulebase*} by substituting the rl-tags to hn-tags in order to provide
	this database for the \RELFUN{}-Interpreter. 

see also: \stmb{hornify-up}
