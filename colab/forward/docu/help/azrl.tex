\azrl:

Format: \stmt{azrl $<$head$>$ $<$body$1>$ ... $<$body$n>$}

Options: $<$head$>$ the conclusion,\\
         $<$body$1>$ ... $<$body$n>$ the premises of an rl clause.

Effect: The rule \stmt{(rl head body1 ... bodyn)} will be inserted at the end of the 
	(possibly empty) database \stmt{*rule-database*}.
        This command is an abbreviation of \stmt{az (rl head body1 ... bodyn)}.

see also: \az, \azup, \consult, \destroy, \replace
