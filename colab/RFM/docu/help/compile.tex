\compile:

Format: \stmt{compile $<$op$>$}

Options: $<$op$>$ an operator

Effect: If $<$op$>$ is given, \stmt{compile} calls \horizon{} to
        compile the whole \RELFUN{} database \stmt{*rfi-database*} horizontally
	and then compiles the operator $<$op$>$ vertically.
	If no argument is given, the whole database \stmt{*rfi-database*} will
	be horizontally and vertically compiled.\\
	There is no difference between the command sequence
	(\horizon, \verti) and the command \stmt{compile}.

see also: \consult, \horizon, \verti
