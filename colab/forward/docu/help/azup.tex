\azup:

Format: \stmt{azup $<$head$>$ $<$body$1>$ ... $<$body$n>$}

Options: $<$head$>$ the conclusion,\\
         $<$body$1>$ ... $<$body$n>$ the premises of an up clause.

Effect: The rule \stmt{(up head body1 ... bodyn)} will be inserted at the end of the 
	(possibly empty) database \stmt{*rule-database*}.
        This command is an abbreviation of \stmt{az (up head body1 ... bodyn)}.

see also: \az, \azrl, \consult, \destroy, \replace
