\stmb{rule-compile}:

Format: \stmt{rule-compile}

Options: none

Effect: Compiles the magic rules (\stmt{*magic-rules*},\stmt{*magic-seeds*}), 
        the bottom-up rules (\stmt{*up-rulbase*}) 
        and the facts (\stmt{*factbase*}) into the FAM.
        


see also: \stmb{eval}, \stmb{magic-query}, \stmb{magic-eval}, \stmb{magic-transform},
