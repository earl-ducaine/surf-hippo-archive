\consult:

Format: \stmt{consult $<$filename$>$}

Options: $<$filename$>$ a stringified or normal pathname

Effect: Global consulting function.
        Loading a hybrid database from file $<$filename$>$ at the end of the
        (possibly empty) databases in \COLAB{} depending on the tags of the
        knowledge items of the file.
        In \FORWARD{} there are \stmt{*rule-database*} and \stmt{*factbase*}.
        If no extension is provided, \COLAB{} extends the filename with \stmt{".rf"}.

see also: \stmb{consult-facts}, \stmb{consult-rules}, \stmb{consult-strategies}, \destroy, 
          \replace
