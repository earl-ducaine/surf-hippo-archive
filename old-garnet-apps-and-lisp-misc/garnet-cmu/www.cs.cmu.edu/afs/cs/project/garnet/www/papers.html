Garnet Papers

Many of the papers written about the Garnet project are here in postscript form. Since starting up a previewer can be slow, we've included plain-text abstracts of each of the papers. To download the postscript (rather than view it), select "Load to Local Disk" from the Options menu (if using Mosaic.)

The file abstracts contains all the abstracts for the papers listed below in plain-text form. There is also a file with the references in Scribe format for the papers listed below. A separate file contains a list of all Garnet papers.
