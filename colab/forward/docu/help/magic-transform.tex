\stmb{magic-transform}:

Format: \stmt{magic-transform $<$goal$1>$ ... $<$goal$n>$}

Options: $<$goal$1>$ ... $<$goal$n>$ the initial seeds for magic sets transformation.

Effect: The database \stmt{*up-rulebase*} will be used to generate new rules wrt to the 
	given goals applying the Magic Sets Transformation method. The new rules
	are stored in the database \stmt{*magic-rules*} and the transformed seeds in 
	\stmt{*magic-seeds*}.

see also: \eval, \stmb{magic-eval}, \stmb{magic-query}
