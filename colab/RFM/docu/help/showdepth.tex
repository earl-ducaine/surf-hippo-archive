\showdepth:

Format: \stmt{showdepth $<$n$>$}

Options: $<$n$>$ a number

Effect: If the system is in trace mode, the output length of the
        conjunction printing will be limited to the number $<$n$>$.
	If this number is 0, no limitation exists. This value is the
	default.

see also: \spy, \nospy
