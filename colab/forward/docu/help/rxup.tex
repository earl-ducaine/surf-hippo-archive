\rxup:

Format: \stmt{rxup $<$head$>$ $<$body$1>$ ... $<$body$n>$}

Options: $<$head$>$ the conclusion,\\
         $<$body$1>$ ... $<$body$n>$ the premises of the up clause.

Effect: The rule \stmt{(up head body1 ... bodyn)} will be removed from the 
	\stmt{*up-rulebase*}, and \stmt{(rl head body1 ... bodyn)} from \stmt{*rule-database*}.
        This command is an abbreviation of \stmt{rx (up head body1 ... bodyn)}.

see also: \rx, \rxrl, \consult, \destroy, \replace
