\stmb{a0fact}:

Format: \stmt{a0fact $<$literal$>$}

Options: $<$literal$>$ the fact

Effect: The fact \stmt{(hn literal)} will be inserted in front of the (possibly empty) 
	database \stmt{*factbase*}. 
	This command is an abbreviation of \stmt{a0 (fact literal)}.

see also: \stmb{a0}, \consult, \destroy, \replace
