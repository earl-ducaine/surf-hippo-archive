\rxrl:

Format: \stmt{rxrl $<$head$>$ $<$body$1>$ ... $<$body$n>$}

Options: $<$head$>$ the conclusion,\\
         $<$body$1>$ ... $<$body$n>$ the premises of an rl clause.

Effect: The rule \stmt{(rl head body1 ... bodyn)} will be removed from the 
	\stmt{*rule-database*}, and \stmt{(hn head body1 ... bodyn)} from the \stmt{*hn-rulebase*}.
        This command is an abbreviation of \stmt{rx (rl head body1 ... bodyn)}.

see also: \rx, \rxup, \consult, \destroy, \replace
