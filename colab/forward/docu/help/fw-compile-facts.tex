\stmb{fw-compile-facts}:

Format: \stmt{fw-compile-facts $<$op$>$ }

Options: $<$op$>$ an operator

Effect: If an operator is given only the matching facts of \stmt{*factbase*} are
	regarded otherwise all facts of \stmt{*factbase*} are vertically compiled 
	into the extended \WAM{}.

see also: \stmb{fw-compile}, \stmb{fw-compile-rules}
