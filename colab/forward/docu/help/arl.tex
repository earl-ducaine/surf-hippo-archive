\stmb{a0rl}:

Format: \stmt{a0rl $<$head$>$ $<$body$1>$ ... $<$body$n>$}

Options: $<$head$>$ the conclusion, and\\
         $<$body$1>$ ... $<$body$n>$ the premises of an rl clause.

Effect: The rule \stmt{(rl head body1 ... bodyn)} will be inserted in front of the 
	(possibly empty) database \stmt{*rule-database*}. 
	This command is an abbreviation of \stmt{a0 (rl head body1 ... bodyn)}.

see also: \stmb{a0}, \stmb{a0up}, \consult, \destroy, \replace
